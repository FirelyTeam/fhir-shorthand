Artifacts Summary

This page provides a list of the FHIR artifacts defined as part of this implementation guide.

  • Structures: Resource Profiles
  • Structures: Extension Definitions

Structures: Resource Profiles

These define constraints on FHIR resources that need to be complied with by conformant implementations

  •  FSH Medication Request Defines constraints and extensions on the MedicationRequest resource for the minimal set of data to query and retrieve prescription information.
  •  FSH Patient Defines constraints and extensions on the patient resource for the minimal set of data to query and retrieve patient demographic information.
  •  FSH Practitioner A profile on the Practitioner resource
  •  FSH Respiratory Rate Respiratory Rate Profile

Structures: Extension Definitions

These define constraints on FHIR data types that need to be complied with by conformant implementations

  •  FSH Birth Sex Extension A code classifying the person's sex assigned at birth as specified by the [Office of the National Coordinator for Health IT (ONC)](https://www.healthit.gov/newsroom/about-onc). This extension aligns with the C-CDA Birth Sex Observation (LOINC 76689-9).
